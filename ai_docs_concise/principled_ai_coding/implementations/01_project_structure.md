### Agent-Ready Project Structure (Concise)

**Why Structure Matters:**
- Predictable, conventional layouts help agents find files, infer relationships, and automate reliably.

**Recommended Layout:**
- `.claude/`: Agent config (commands, hooks, settings)
- `specs/`: Versioned feature plans
- `logs/`: Observability data from hooks
- `src/`: Application code
- `tests/`: Test code
- `docs/`: Project and AI coding documentation

**Signal:**
A clear, versioned structure enables reliable, scalable agentic development and automation.
