# Claude Code Quickstart (Concise)

**Getting Started:**
- Install via `npm install -g @anthropic-ai/claude-code` or native script.
- Start in any project directory: `claude`
- Ask questions or give tasks in natural language (e.g., "what does this project do?", "add a hello world function").
- Claude Code reads files as needed and always asks before making changes.

**Key Commands:**
- `claude` — Start interactive mode
- `claude -p "query"` — One-off prompt
- `/help` — List available commands
- `/clear` — Clear conversation

**Signal:**
Claude Code is your AI pair programmer—install, start, and interact naturally to accelerate development.
