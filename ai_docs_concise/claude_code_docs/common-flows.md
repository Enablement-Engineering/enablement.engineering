# Claude Code: Common Workflows (Concise)

Claude Code supports these core agentic workflows:

- **Codebase Understanding:** Ask for overviews, architecture, and key components.
- **Code Search:** Locate files and trace feature flows.
- **Bug Fixing:** Share errors, get fix suggestions, and apply changes.
- **Refactoring:** Identify legacy code, get modern recommendations, and apply/test refactors.
- **Subagents:** Delegate tasks to specialized AI agents for review, debugging, or testing.
- **Testing:** Find untested code, generate scaffolding, add edge cases, and run tests.
- **Pull Requests:** Summarize changes, generate PRs, and add testing details.
- **Documentation:** Identify undocumented code, generate and review docs.
- **Image Analysis:** Use images for context, error analysis, or UI suggestions.
- **File/Directory Reference:** Use @ to include files or directories in context.
- **Extended Thinking:** Trigger deep reasoning for complex tasks.
- **Session Management:** Resume or continue previous conversations.
- **Parallel Worktrees:** Run multiple Claude sessions in isolated git worktrees.
- **Unix Utility Mode:** Integrate Claude into scripts for linting, summarization, or code review.
- **Custom Slash Commands:** Create reusable, parameterized prompts for team workflows.

**Signal:**
These workflows enable efficient, repeatable, and collaborative agentic development with Claude Code.
