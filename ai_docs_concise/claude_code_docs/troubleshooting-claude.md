# Claude Code Troubleshooting (Concise)

**Purpose:**
Diagnose and resolve common issues with Claude Code operation, configuration, or integration.

**Key Steps:**
- Check installation, API keys, and permissions.
- Use `/doctor` for health checks.
- Review logs and error messages for clues.
- Consult documentation or community for persistent issues.

**Signal:**
Troubleshooting is about verifying setup, permissions, and configuration, and using built-in tools and docs to resolve problems quickly.
