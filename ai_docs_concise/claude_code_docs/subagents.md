# Claude Code Subagents (Concise)

**Purpose:**
Subagents are specialized AI agents within Claude Code, each with its own tools, prompts, and system instructions, enabling parallelism and task delegation.

**Key Concepts:**
- Use subagents for code review, debugging, testing, or domain-specific tasks.
- Create/manage via `/agents` command.
- Limit tool access and define custom prompts per subagent.

**Signal:**
Subagents enable scalable, modular, and parallel agentic workflows, letting you delegate complex or specialized tasks to the right AI expert.
