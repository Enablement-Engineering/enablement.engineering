# Claude Code Interactive Mode (Concise)

**Purpose:**
Interactive mode is the default REPL for Claude Code, supporting conversational, agentic development with keyboard-driven controls.

**Key Controls:**
- `Ctrl+C`: Cancel input/generation
- `Ctrl+D`: Exit session
- `Up/Down`: Command history
- `/`: Slash commands (see `/help`)
- `#`: Add to CLAUDE.md memory
- Multiline: `Option+Enter` (macOS), `Shift+Enter` (custom)
- Vim mode: `/vim` for modal editing

**Features:**
- Command history per directory
- Quick access to slash commands and memory
- Supports both iterative and plan-driven workflows

**Signal:**
Interactive mode is the primary interface for agentic coding, enabling fast, keyboard-centric collaboration with Claude Code.
